README - マイポータル

個人ローカル環境で使うことを想定した、シンプルなポータルサイトです。

JSONファイルで手軽にリンク集を作成できます。

その他用途に合わせて自由にカスタマイズが可能です。

特徴

  • 主にWeb開発者が日々の開発で便利に使えることを目指しています。
  • 外部ライブラリを一切使用していません。
  • リンク集はJavaScriptを使用するので、ローカル環境でWebサーバーを起動して利用することを想定しています。
  • カスタマイズにはHTML, CSS, JavaScriptの知識が必要になります。デフォルトで入っているWeb開発リンク集をお役立てください。

導入

マイポータルのGitHubページから、URLをコピーしてgit cloneするか、Download Zipして展開してください。

git cloneした場合は、ローカルで自分用のブランチを作成して管理すると便利です。

カスタマイズ

ホームページを変更する

デフォルトのホームページを変更するか、他の既存のHTMLと置き換えることができます。index.htmlでiframeに指定するソースをlink/index.htmlにすれば、リンク集をホームページにすることもできます。

リンクの項目を追加・削除する

対応するJSONファイルを編集してください。

リンクのグループを追加・削除する

idを指定したdivタグと、id名に対応する名前のjsonファイルを作成します。

グループを削除する時は、対応するdivタグとjsonファイルを削除して下さい。

リンクのHTMLをカスタマイズする

リンクのHTMLは、リンクを作成するdivタグ以外は特に制限がありません。 link-groups-containerを複数置くこともできます。 web-dev.htmlのように、別のHTMLを追加することも可能です。 他にも開発中のメモやコメント・画像を追加するなど、自由にカスタマイズしてください。

その他のWebリソースを組み込む

index.htmlのleft-contentsにリンクを追加すれば、ローカル環境で動作するWebページ・Webアプリケーションをマイポータルの一部として閲覧・利用することができます。
